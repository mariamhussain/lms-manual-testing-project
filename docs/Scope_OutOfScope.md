# Scope & Out of Scope – LMS

## ✔ In Scope
- User login & registration
- Viewing course catalog
- Searching for courses
- Enrolling in free courses
- Making payments for paid courses
- Watching lessons
- Viewing progress

## ❌ Out of Scope
- Instructor course creation
- Admin curriculum management
- Push notifications
- Mobile app
- Gamification features
